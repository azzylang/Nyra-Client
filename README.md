# Contribution

- Make a pullrequest with the updated/new code
- After that we will look at the code and after discussing it, we will accept the request
- Then we will add you to the contributor list and optionally give you the role on our discord server

## Suggestions or Bugs

- To submit Bug Reports or Suggestions please join the official discord server and submit your report/reports or suggestions

# Build Instructions

- Clone this repository (via `git clone https://github.com/constttdev/nyra-client` or clicking on **Code** and then on **Download ZIP**)
- Then run `./gradlew build`
